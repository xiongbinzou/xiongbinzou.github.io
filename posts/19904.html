Python虚拟环境的使用

发表于2022-08-07|更新于2022-08-12|Python
|字数总计:132|阅读时长:1分钟

python虚拟环境的主要目的是为Python项目创建一个隔离的环境，意味着每个环境都可以拥有自己的依赖项。

1.安装virtualenv

pip install virtualenv

2.创建虚拟环境并指定Python版本

virtualenv venv --python=python3.8

3.进入虚拟环境

source bin/active

4.退出虚拟环境

deactive

5.删除虚拟环境

rm -r venv

6.生成requirements.txt文件

方法一:

pip freeze > requirements.txt

方法二:

pip install pipreqs

pipreqs ./ --encoding=utf-8 --force

7.使用requirements.txt文件

pip install -r requirements.txt
